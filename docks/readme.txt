Welcome to NeuroNest!
NeuroNest is an accessible note-taking app. This app is designed to be clean, fast, and screen reader friendly, with full support for creating/using keyboard themes, and intuitive file management.

Features:
Create, load, and edit notes in plain text format.
Rename or delete notes easily.

How to use.
Launch the app.
Choose “Create note” or “Load note” from the menu.
Start typing your note.
Use the “Save” or “Save As” buttons when you're ready to finish writing your thoughts.
Press F1 in any input box to cycle through the available keyboard themes.
You can place new keyboard theme folders in sounds/typing themes.
